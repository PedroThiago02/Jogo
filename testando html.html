Sobre cada tag:

A maioria das tags de abertura precisam de uma tag de fechamento, com exceção de algumas, como a tag "<br>" (para quebra de parágrado, sendo utilizado como "gambearra", visto que o indicado é fazer as quebras de parágrafo no CSS). As tags mais conhecidas em um html são as tags de <html> (que serve para iniciar um codigo html), a tag <head> (é o cabeçalho de um codigo html), onde podemos colocar tags como <title> (titulo do codigo) e conexões com o CSS e JavaScript (que servem para deixar o código mais dinamico e bonito). Já o <body>, é utilizado como o "corpo" do código e é nele que temos a parte visual do site. Dentro do <body> temos várias outras tags:


<h1>: Titulo do texto

<h2>: Subtitulo 1

<h2>: Subtitulo 2

<h4>: Subtitulo 3

<h5>: subtitulo 4
<:h6>: subtitulo 5
